A simple short code for inserting jScrollPane content in any WordPress post or page.

== Description ==

This plugin will embed a scrolling content section with jScrollPane.

You can get more information on this plugin [VR jScrollPane Shortcode](http://blog.pixelthemes.com/plugin/vr-jscrollpane-shortcode-plugin-for-wordpress/ "VR jScrollPane Shortcode Pluign URL").
